Задание c использованием LinQ:
Дана последовательность непустых строк.
Получить последовательность символов, которая определяется следующим образом:
если соответсвующая строка исходной последовательности имеет нечетную длину, то в качестве символа берется первый символ этой строки;
в противном случае берется последний символ строки.
Отсортировать полученные символы по убыванию их кодов.

Запуск программы:
1.Вводите число строк(непустых) которые вы хотите ввести.
2.Программа выполняется согласно условиям выше.

Запуск можно осуществлять с помощью bat-файлов:
1.run.bat - запуск программы.
2.build.bat - сборка программы.
