Дано: 
Строка из 30 слов.Соседние слова разделены запятой.В конце строки точка.
Задание:
Вывести все слова и число их вхождений в строку.
Вывод: 
Длина строки.
Вывод слов осуществляется начиная с первого и по 30-ое + количество их вхождений в строку.

Пример вывода с 5-ью словами:
Дана строка:
Строка = "a,b,c,d,d,d."
Вывод:
1)a: 1
2)b: 1
3)c: 1
4)d: 3

Запус программы можно осуществлять с помощью .bat файлов.
1)buil.bat делает сборку проекта.(рекомендуется перед запуском)
2)run.bat запускает проект.
