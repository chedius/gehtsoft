Программа возвращает все строчные русские буквы из заданного текста в алфавитном порядке.
Она может выполнить это со сгенериррованым текстом содержащим латиницу, цифры, знаки, русские строчные и прописные буквы.
Есть функционал для проверки работаспособности отдельных функций.
Для сборки программы можно воспользоваться пакетным файлом build.bat в корневом каталоге программы.
Для запуска можно использовать пакетный файл run.bat , находящийся также в корневом каталоге программы.
